Nu skall du skapa kursens git repo lokalt i din terminal under katalogen `dbwebb-kurser`, och koppla det till repot som du nyligen skapade på GitHub.

## Gå till katalogen dbwebb-kurser

Börja med att gå till din arbetskatalog för kurserna, den som du döpte till `dbwebb-kurser`.

```bash title="Gå till katalogen dbwebb-kurser"
# Gå till hemmakatalogen
cd

# Gå till dbwebb-kurser
cd dbwebb-kurser
```

Du kan göra samma sak i ett kommando och här refererar `~` till din hemmakatalog.

```bash title="Gå till katalogen dbwebb-kurser, i ett kommando"
cd ~/dbwebb-kurser
```

Använd kommandot `pwd` och `ls` för att försäkra dig om att du är på rätt plats i filsystemet.


## Skapa kursrepot som ett lokalt git repo

Nu skall vi skapa ett lokalt kursrepo från det repo du nyss skapade på GitHub.


### Klona ditt GitHub repo till ett lokalt repo

Nu ska vi skapa en lokal kopia av ditt GitHub repo genom att klicka på "Code" och kopiera SSH-länken.

<Figure 
    src={ghnew3}
    caption="Klona ditt repot på GitHub."

```bash title="Kommandon för att klona ditt kursrepo"
git clone git@github.com:bth-csharp/csharp-<akronym>.git
```

Om din akronym är `abcd25` så ser kommandot ut så här:  
`git clone git@github.com:bth-csharp/csharp-abcd25.git` 


### Kolla ditt lokala Git repo

Du kan använda kommandot `pwd` och `ls` för att kontrollera vilken katalog du är i och vad den innehåller. Använder du `ls -la` får får du detaljer om filer och kataloger och det visas även dolda kataloger som börjar med en punkt `.`.

Så här ser ditt Git repo ut i terminalen:

<Figure

Nu kan du köra följande Gitkommandon för att status, kolla vilka brancher som finns i ditt kursrepo.

```bash title="Gitkommandon för att kolla i ditt lokala kursrepo"
cd csharp-<akronym>
git status
git branch
git branch -r
git log
```

Så här kan det se ut i terminalen:

`git branch -r` visar de brancher du har i ditt GitHub repo, de behöver du inte lokalt om inlämningen går som den ska.
Du skall jobba i branchen `main`och här kommer du att placera all kod du skriver.

De andra fyra tomma branches ska användas vid inlämning av kursens tre uppgifter. Den sista är för att kolla att din utvecklingsmiljö är kopplad till Canvas.


:::caution
Om något går fel så kan du radera din katalog och försöka igen.

```bash title="Så här raderar du ditt skapade repo om du vill börja om från början."
# Change <dirname> to the name of your course repository name,
# for example csharp-abcd25.
cd ~/dbwebb-kurser
rm -rf <dirname>
```

Kör därefter kommandona som skapar ditt kursrepo igen, men kör dem gärna ett efter ett så blir det lättare att se vilket kommando som går fel.
:::
