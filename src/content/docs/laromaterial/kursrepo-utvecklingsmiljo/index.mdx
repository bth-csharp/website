## Förutsättning

För att genomföra denna guiden behöver du tillgång till ditt kursrepo som du skapar. I denna guiden kommer jag att använda kursrepot `csharp-abcd25` som exempel.

Innan du börjar jobba i guiden så öppnar du en terminal och flyttar till katalogen för ditt kursrepo.

```bash
# ersätt abcd25 med din akronym 

cd dbwebb-kurser/csharp-abcd25
```

## Guidens innehåll

I guiden får du hjälp med:

1. Lägga till konfigurationsfilen `.editorconfig` för inställningar i vscode.

1. Använda git-kommandon som add, commit, push, pull, status och log för att jobba med ditt Git-repo.

1. Lägga till filen `.gitignore` för att ignorera vissa filer och kataloger från att inkluderas i ditt repo.
