Task är ett byggverktyg och "task runner". Det är ett verktyg som förenklar processen att köra olika kommandon i terminalen. Det är ett verktyg som kan hjälpa oss att automatisera uppgifter som vi annars skulle behöva skriva in manuellt varje gång. Du kan läsa mer på dess [webbplats](https://taskfile.dev/).

Vi ska främst använda det för att förenkla att ladda ner olika filer i kursen.

## Installera Task

Det finns många olika sätt att installera Task och här presenteras ett sätt.

### Ubuntu (Windows/Linux)


```console title="Installera Task med Snap."
sudo snap install task --classic
```

### Mac

```console title="Installera Task med Homebrew."
brew install go-task
```

### Verifiera installationen av Task

```console title="Verifiera installationen av Task"
task --version
// ger i mitt fall utskriften: 3.43.3
task --help // ger hjälp
```

## Kör task

Du kan kolla vilka kommandon som finns med följande kommando.

```console title="Lista kommandon i Taskfile."
task --list
```

`task` är kommandot för att köra Task och `--list` är en flagga som listar alla kommandon som finns i Taskfile.

För tillfället finns följande kommandon tillgängliga:

```
task: Available tasks for this project:
* download-lab:               NOT READY - DO NOT USE!
* download-taskfile:          Downloads the Taskfile from the template repository. Run with 'task download-taskfile'
* download-tests:             Downloads tests for the specified kmom, e.g. kmom01. 'task download-tests -- kmom01'
* download-workflow:          NOT READY - DO NOT USE!
```
